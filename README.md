<h2>How to <b>very fast</b> make page like this?</h2>

Create new public github repository with this name:<br>
`username.github.io`<br>
With your username<br>
***
Then copy files from this repository into yours<br>
Open `script.js`<br>
Change data in the `locale` , `images` and `links` to your locale, files and links
********************************
Open settings of your repository, and enable GitHub Pages<br>
Wait a little bit, and open page `username.github.io` with your username in your browser.
